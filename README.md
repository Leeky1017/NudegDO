# NudgeDO

**可调节强度的 AI 规划教练** - 帮用户"启动"而非"记录"

## 产品理念

- **AI 主动提问**：定时推送，引导用户规划明日任务
- **多轮追问**：可调节追问深度（1-5次），直到任务安排清晰
- **ADHD 增强模式**：原子级任务拆解 + 启动仪式 + "卡住了"按钮

## 核心差异

| 传统 TODO | NudgeDO |
|-----------|---------|
| 被动记录 | 主动推动 |
| 用户自己安排 | AI 引导规划 |
| 记录数量 | 完成率 |

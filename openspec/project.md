# NudgeDO - 项目元信息

## 基本信息
- **项目名称**: NudgeDO
- **项目类型**: AI TODO 应用
- **GitHub**: https://github.com/Leeky1017/NudegDO.git

## 项目定位
通过 AI 主动追问，帮助用户将模糊想法转化为清晰可执行的任务。

## 核心特性
1. **双模式设计**: OnMe（快速记录）vs Nudge（AI 追问）
2. **轮次制追问**: 1-2 轮，每轮 2-3 个问题
3. **AI 人格系统**: Coach / Buddy / Drill / Zen
4. **ADHD 增强模式**: 可选的额外支持功能

## 技术选型
- **LLM**: GLM4.5 AIR（首选免费模型）
- **前端**: React + TypeScript + TailwindCSS
- **后端**: 待定

## 工作流
采用 OpenSpec + Rulebook + GitHub 三体系交付。

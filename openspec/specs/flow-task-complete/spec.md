# FLOW-004: 任务完成流程

## 概述
用户点击 checkbox 切换任务完成状态。

## 需求

### 功能需求
- [ ] R1: 点击 checkbox 切换 completed 状态
- [ ] R2: 完成状态下标题显示删除线
- [ ] R3: checkbox 显示勾选动画
- [ ] R4: 更新任务计数（仅统计未完成）
- [ ] R5: 可再次点击取消完成状态

### 视觉反馈
- [ ] V1: checkbox 选中时背景变绿 + 白色勾
- [ ] V2: 标题添加删除线 + 颜色变灰
- [ ] V3: 过渡动画 0.2s ease

## 验收标准
1. 点击可切换状态
2. 视觉反馈正确
3. 任务计数更新

# FLOW-002: Nudge 模式触发流程

## 概述
用户通过 N 键或 N 按钮激活 Nudge 模式，触发 AI 追问。

## 需求

### 功能需求
- [ ] R1: 点击 N 按钮切换 Nudge 模式
- [ ] R2: Ctrl+N 快捷键切换 Nudge 模式
- [ ] R3: 激活时 N 按钮显示呼吸动画
- [ ] R4: 如果输入框有内容，立即展开追问面板
- [ ] R5: 如果输入框为空，仅切换状态，等待输入后再展开
- [ ] R6: 再次点击 N 按钮可退出 Nudge 模式

### 流程图
```
用户点击 N 按钮 / Ctrl+N
      ↓
  切换 isNudgeMode 状态
      ↓
  更新 N 按钮视觉状态
      ↓
  [有输入内容?]
      ↓ Yes              ↓ No
  展开追问面板      等待用户输入
      ↓                   ↓
  生成追问问题      用户输入后展开
```

## 验收标准
1. N 按钮点击可切换状态
2. Ctrl+N 快捷键有效
3. 有内容时立即展开面板
4. 无内容时等待输入

# LOGIC-001: 轮次制追问系统

## 概述
Nudge 模式采用轮次制追问，每轮包含 2-3 个问题打包提问。

## 需求

### 功能需求
- [ ] R1: 支持 1-2 轮追问（用户可配置）
- [ ] R2: 每轮生成 2-3 个相关问题
- [ ] R3: 问题围绕：时间、障碍、拆解、提醒
- [ ] R4: 第 1 轮：基础信息收集
- [ ] R5: 第 2 轮（可选）：深度确认和备选方案

### 问题维度
| 维度 | 第 1 轮 | 第 2 轮 |
|------|---------|---------|
| 时间 | 什么时候开始？需要多久？ | 备选时间？ |
| 障碍 | 可能的阻碍因素？ | 如何应对？ |
| 拆解 | - | 是否需要拆分子任务？ |
| 提醒 | 需要提醒吗？ | 提醒方式偏好？ |

### 配置项
```typescript
interface RoundConfig {
  maxRounds: 1 | 2;           // 最大轮数
  questionsPerRound: 2 | 3;   // 每轮问题数
  granularity: 'coarse' | 'medium' | 'fine';  // 拆解粒度
}
```

## 验收标准
1. 默认 1 轮，每轮 2-3 个问题
2. 问题覆盖时间、障碍等维度
3. 配置可调整轮数

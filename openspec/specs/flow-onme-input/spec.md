# FLOW-001: OnMe 模式输入流程

## 概述
OnMe 模式是默认的快速记录模式，用户输入任务后按 Enter 直接存入。

## 需求

### 功能需求
- [ ] R1: 用户在输入框输入任务文本
- [ ] R2: 按 Enter 键直接创建任务（不触发 Nudge）
- [ ] R3: 点击提交按钮也可创建任务
- [ ] R4: 创建后清空输入框
- [ ] R5: 新任务添加到列表顶部
- [ ] R6: 空输入不允许提交

### 流程图
```
用户输入文本 → Enter/点击提交
      ↓
  检查是否为空
      ↓
  [非空] → 创建 Task 对象
      ↓
  添加到任务列表顶部
      ↓
  清空输入框
      ↓
  更新任务计数
```

## 验收标准
1. Enter 键可直接提交
2. 空输入时提交无效
3. 新任务出现在列表顶部
4. 输入框提交后清空

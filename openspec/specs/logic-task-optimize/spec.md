# LOGIC-003: 任务优化

## 概述
根据用户回答优化任务标题，提取时间和时长信息。

## 需求

### 功能需求
- [ ] R1: 从用户回答中提取关键信息
- [ ] R2: 优化任务标题（更具体、可执行）
- [ ] R3: 提取时间信息（开始时间）
- [ ] R4: 提取时长信息
- [ ] R5: 生成提醒时间（开始前 5 分钟）

### 优化示例
```
原始输入: "写一份行业报告"
用户回答: "新能源行业，给投资人看。明天下午2点，预计3小时。"

优化后:
- 标题: "写一份新能源行业调研报告（投资人向）"
- 时间: 明天 14:00
- 时长: 3h
- 提醒: 明天 13:55
```

### LLM Prompt 模板
```
原始任务：{original_task}
用户补充信息：{user_response}

请提取并返回 JSON：
{
  "optimized_title": "优化后的任务标题",
  "start_time": "HH:MM 或 null",
  "duration": "Xh/Xm 或 null",
  "reminder_offset": 5  // 提前几分钟提醒
}
```

## 验收标准
1. 标题优化更具体
2. 时间信息正确提取
3. 提醒时间自动计算

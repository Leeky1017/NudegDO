# LOGIC-002: 问题生成策略

## 概述
根据任务文本和人格类型生成追问问题。

## 需求

### 功能需求
- [ ] R1: 分析任务文本提取关键信息
- [ ] R2: 根据人格类型调整问题语气
- [ ] R3: 避免重复问题
- [ ] R4: 支持模板问题（离线可用）
- [ ] R5: 支持 LLM 动态生成（在线增强）

### 问题模板（按人格）

**Coach 教练**：
1. "这个任务的具体目标是什么？"
2. "你打算什么时候开始？需要多长时间？"
3. "有什么可能阻碍你的因素？"

**Buddy 伙伴**：
1. "能跟我说说这个任务是关于什么的吗？"
2. "你觉得什么时候做比较舒服？大概要多久呢？"
3. "有什么我可以帮你提前准备的吗？"

### LLM Prompt 模板
```
你是一个{persona}风格的任务规划助手。
用户想要完成：{task_text}

请生成 {count} 个追问问题，帮助用户明确：
1. 具体目标和范围
2. 时间安排
3. 可能的障碍

要求：
- 语气符合{persona}风格
- 问题简洁明了
- 不要重复
```

## 验收标准
1. 模板问题可离线使用
2. LLM 生成问题符合人格风格
3. 问题不重复

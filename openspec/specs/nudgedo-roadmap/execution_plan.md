# NudgeDO Roadmap — Execution Plan

本文件用于表达“硬依赖”（必须先合并的 Issue）。`scripts/agent_pr_preflight.sh` 会解析本文件并在 PR 前给出阻塞提示。

## 格式（1 行 1 个 Issue）

在同一行使用以下任一关键词即可声明依赖（支持中英混写）：
- `hard depends on: #12 #34`
- `MUST after #12 #34`
- `依赖 #12 #34`

示例：

```text
#N <short-title> (hard depends on: #M #K)
```
